noVNC browser session

Tor session test

This launches a disposable Chromium desktop inside the container, routes it through Tor, and exposes the screen in this tab through noVNC.

Open browser tab Open manually
Local URL after running Docker: http://localhost:6080
Default start page: https://check.torproject.org. Override with the URL environment variable.
Do not leave this public without access controls. Anyone with the URL can use the session unless you set a VNC password or protect the service.
